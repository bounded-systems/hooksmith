## Basic CLI Setup

Basic CLI setup example.

### Dependencies

- anyhow
- clap
- serde

### Code

```rust
use anyhow::Result;
use clap::Parser;

#[derive(Parser)]
struct Cli {
    #[arg(long)]
    verbose: bool,
}

fn main() -> Result<()> {
    let cli = Cli::parse();
    println!("Verbose: {}", cli.verbose);
    Ok(())
}
```
