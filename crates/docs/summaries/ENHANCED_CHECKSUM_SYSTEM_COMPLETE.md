### ✅ **Validation Workflow**

#### **1. Registry-Based Validation**
```rust
// Load registry
let registry = ChecksumRegistry::load()?;

// Validate all files
let report = registry.validate_all()?;

// Check results
if report.is_successful() {
    println!("✅ All files validated successfully");
} else {
    for (path, error) in &report.errors {
        eprintln!("❌ {}: {}", path, error);
    }
}
```

#### **2. Individual File Validation**
```rust
// Find file in registry
if let Some(entry) = registry.find_by_path("README.md") {
    let is_valid = registry.validate_file(entry)?;
    if is_valid {
        println!("✅ README.md checksum matches");
    } else {
        println!("❌ README.md has been modified");
    }
}
```

#### **3. Checksum Updates**
```rust
// Update all checksums in registry
let update_report = registry.update_all_checksums()?;

// Save updated registry
registry.save()?;

// Report results
println!("Updated: {} files", update_report.updated.len());
println!("Unchanged: {} files", update_report.unchanged.len());
```
