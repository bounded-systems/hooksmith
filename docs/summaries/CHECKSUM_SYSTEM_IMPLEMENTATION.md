#### **2. Checksum Utility Module (`xtask/src/checksum.rs`)**

**Core Functions**:
```rust
/// Compute a deterministic checksum for file content
pub fn compute_file_checksum(content: &str) -> String {
    let mut hasher = Sha256::new();
    hasher.update(content.as_bytes());
    let result = hasher.finalize();
    
    // Return first 8 characters of hex string
    format!("{:x}", result)[..8].to_string()
}

/// Extract content without header lines for checksum computation
pub fn extract_content_without_header(content: &str, header_lines: &[String]) -> String {
    let lines: Vec<&str> = content.lines().collect();
    let header_count = header_lines.len();
    
    if lines.len() <= header_count {
        return String::new();
    }
    
    // Skip header lines and return the rest
    lines[header_count..].join("\n")
}

/// Generate a complete header with checksum
pub fn generate_header_with_checksum(
    file_type: &str,
    content_without_header: &str,
) -> String {
    let checksum = compute_file_checksum(content_without_header);
    
    match file_type {
        "md" => format!(
            "<!-- @generated by xtask gen-files --file-type=md -->\n<!-- @checksum: {} -->\n",
            checksum
        ),
        "toml" => format!(
            "# @generated by xtask gen-files --file-type=toml\n# @checksum: {}\n",
            checksum
        ),
        // ... other file types
    }
}

/// Validate a file's checksum against its content
pub fn validate_file_checksum(
    file_path: &Path,
    file_type: &str,
) -> Result<bool> {
    let content = fs::read_to_string(file_path)?;
    
    // Extract the expected checksum from the header
    let expected_checksum = extract_checksum_from_header(&content, file_type)?;
    
    // Compute checksum of content without header
    let content_without_header = extract_content_without_header(&content, &get_header_template(file_type));
    let actual_checksum = compute_file_checksum(&content_without_header);
    
    Ok(expected_checksum == actual_checksum)
}

/// Write a file with generated header and checksum
pub fn write_file_with_checksum(
    file_path: &Path,
    file_type: &str,
    content: &str,
) -> Result<()> {
    let header = generate_header_with_checksum(file_type, content);
    let full_content = format!("{}{}", header, content);
    
    fs::write(file_path, full_content)?;
    Ok(())
}
```

### ✅ **Validation Workflow**

1. **Read File Content**: Load the entire file into memory
2. **Parse Header**: Extract the checksum from the header lines
3. **Strip Header**: Remove header lines from content
4. **Compute Hash**: Calculate SHA-256 hash of remaining content
5. **Extract Checksum**: Take first 8 characters of hex string
6. **Compare**: Compare computed vs. extracted checksum
7. **Report Result**: Return validation status
